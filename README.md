# JSI Website (Next.js + Tailwind)
## Quick start
1. Install dependencies:
   ```
   npm install
   ```
2. Run dev server:
   ```
   npm run dev
   ```
3. Visit http://localhoRun st:3000
## Notes
- Replace Google Analytics ID and Search Console meta in app/layout.tsx if you have custom ones.
- Replace Formspree endpoint in /app/contact/page.tsx if you have a custom one.
